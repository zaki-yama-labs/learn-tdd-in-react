Learn TDD in React
==================

[Learn TDD in React](http://learntdd.in/react/)


### Usage

#### Run Locally

```
$ yarn
$ yarn start
```

#### Deploy to Heroku

After installing [Heroku CLI](https://github.com/heroku/cli), run

```
$ heroku apps:create [your-app-name]
$ git push heroku master
$ heroku open
```
